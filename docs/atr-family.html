Auto Landing Lights

  • Automatically turn the landing lights and other lights off/on when passing through a certain altitude.
  • You must ARM it by clicking on the widget, and it will turn orange. Then, while the airplane crosses the specified altitude, the actions will be triggered accordingly.
  • Executing the Before Takeoff widget will arm this automatically. Don't forget to arm it again before descending.
  • The altitude reported by MSFS might be different from the one shown on the PFD, so giving some margin is recommended.

Set Altitude

  • The widget shows the current MCP altitude.
  • Scroll on the widget to change the MCP altitude.
  • Use OTTO to set (and intervene) the altitude to a value. Some example commands would be "alt 3000", "fl 320", "fl 200 intv", "alt intv", etc.

Set Minimums

  • Scroll on the widget to change the minimums value.
  • Use OTTO to set the minimums to a value. Some example commands would be "mins 200", "baro 360", "radio 200", "mda 240", etc.
